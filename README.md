# DictoKey - AI Voice Keyboard for Android

**[dictokey.com](https://dictokey.com)** | **[Get it on Google Play](https://play.google.com/store/apps/details?id=com.dictokey.app)**

DictoKey is an AI-powered voice keyboard for Android that replaces your default keyboard with real-time speech-to-text. Powered by OpenAI Whisper, it supports 52 languages with dictation, translation, and AI rewrite -- all processed locally on your device.

## Key Features

- **Voice dictation** in 52 languages with Whisper AI accuracy
- **Real-time translation** -- speak in one language, type in another
- **AI rewrite** -- fix grammar, change tone, summarize
- **Works everywhere** -- any app, any text field
- **No account required** -- install and start dictating immediately
- **GDPR compliant** -- audio processed on-device, no data collection
